# Alteração: Consolidação de Dados na Aba 7 para Perfis de Manifestação

## 1. Contexto e Motivação
A pedido do cliente, todos os perfis de manifestação (escalões gerenciais que interagem acima de "Chefia" e a própria "Chefia") devem interagir estritamente através da **Aba 7 (Manifestações)**. Eles não precisam ver as abas técnicas soltas no menu principal para evitar poluição visual e desvio do fluxo de assinaturas. Ao invés disso, os insumos técnicos das abas anteriores devem estar consolidados e incorporados na própria Aba 7 em formato de resumos expandíveis (acordeões).

## 2. Solução Implementada
1. No `ProcessoController.php`, o método `getAbasDoPerfil` foi revertido para que os perfis de manifestação (ex: `Chefia`, `Coordenação`, `Superintendência`, etc.) visualizem exclusivamente a matriz `[7]` na navegação.
2. No `ProcessoController@show`, quando a requisição recai sobre a aba 7, adicionamos o carregamento dos pacotes de dados das abas anteriores (`$dados1`, `$dados2`, `$dados3`, `$rips`, `$cadastros`) via extração do modelo `$foco`, que são injetados na view.
3. No template `aba7.blade.php`, emulamos o comportamento bem-sucedido de apresentação da view de `historico`. Inserimos quatro blocos de acordeões no início do formulário de despacho, que embutem (via `iframe`) as telas prototipadas de resumo: Requerimento (1a), RIP (1b), Diagnóstico (2) e Análise de Proposta (3).
4. Essa abordagem reaproveita as instâncias estáticas que usam localStorage, alimentadas nativamente pelo frontend original das abas.

**Estado Anterior (Antes):**
```php
    private function getAbasDoPerfil(string $perfil): array
    {
        $map = [
            'Equipe Destinação'   => [1, 3],
            'Equipe Caracterização' => [2],
            'Chefia'              => [1, 2, 3, 7],
            'Coordenação'         => [1, 2, 3, 7],
            // ...
```

**Estado Novo (Depois):**
```php
    private function getAbasDoPerfil(string $perfil): array
    {
        $map = [
            'Equipe Destinação'   => [1, 3],
            'Equipe Caracterização' => [2],
            'Chefia'              => [7],
            'Coordenação'         => [7],
            // ...
```

**Plano de Rollback / Desfazer:**
Desfazer o update no `getAbasDoPerfil`, removendo a restrição e retornando aos arrays expandidos. Remover no método `show` as condicionais de `if ($aba == 7 && $foco)` que buscam os `$dados1,2,3`.
