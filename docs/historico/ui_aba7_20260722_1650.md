# Atualização de UI - Aba 7 (Caixas de Opção e Selo de Origem)

**Data/Hora**: 22 de Julho de 2026, 16:50
**Arquivo Alterado**: `resources/views/processos/abas/aba7.blade.php`

## O que foi feito?
- Adição de CSS para `.decl-origem-stamp` e `.decl-section-title`.
- Refatoração do formulário da Aba 7 para englobar todas as abas (Chefia, Coordenação, Superintendência, CDE, etc) dentro do loop dinâmico `@foreach($secoes as $chave => $s)`, reduzindo código duplicado.
- Inserção dinâmica do selo de origem baseando-se no timestamp da aba/assinatura anterior correspondente.
- Atualização da estrutura HTML dos *radio buttons* da Superintendência para usar a estrutura compátivel com o CSS avançado (`input` + `label`).

## Estado Anterior (Antes)
O código antigo possuía um bloco fixo FORA do loop para a aba da Superintendência que utilizava a seguinte estrutura de radio:
```html
<label class="decl-opcao-btn-label">
    <input type="radio" name="sup_interesse" value="consistente"> Evidenciado de forma consistente
</label>
```
Isso impedia o CSS `.decl-opcao-btns > input[type="radio"]:checked + .decl-opcao-btn-label` de ser aplicado porque o input não era irmão da label.

## Estado Novo (Depois)
```html
<div class="decl-opcao-btns">
    <input type="radio" name="sup_interesse" id="sup-int-1" value="consistente">
    <label class="decl-opcao-btn-label" for="sup-int-1">
        <span class="opcao-icone">✔</span>
        <span>Evidenciado de forma consistente</span>
    </label>
</div>
```
E toda a lógica dos formulários pendentes (seja Padrão, Superintendência ou CDE) foi injetada elegantemente num grande bloco `@if / @elseif / @else` dentro do `acordeao-corpo` do foreach global das seções, ativando automaticamente o carimbo de origem:
```blade
@php $origemTextoAtiva = getOrigemTexto($chave, $processo, $dados); @endphp
@if($origemTextoAtiva)
<div class="decl-origem-stamp">
    {{ $origemTextoAtiva }}
</div>
@endif
```

## Plano de Rollback / Desfazer
Para reverter completamente estas alterações de design:
1. Abra `resources/views/processos/abas/aba7.blade.php`.
2. Remova as classes CSS `.decl-origem-stamp` e `.decl-section-title`.
3. Desfaça a refatoração do loop `secoes`, retirando o bloco `@if($chave === 'superintendencia')` e recolocando-o **fora** do foreach (entre o final das abas concluídas e o início do loop da ativa).
4. Retire as chamadas à função `getOrigemTexto()`.
